# Keithley 6487 Python Control

Python scripts for:
- Live I–V sweeps
- Constant-bias I–t measurements
- CZT and high-resistance detector characterisation

## Requirements
- Python 3.9+
- pyvisa
- numpy
- matplotlib

## Tested with
- Keithley 6487 Picoammeter / Voltage Source
- NI-VISA on Windows

## Notes
All scripts include:
- Safe shutdown
- Compliance limiting
- Live plotting
